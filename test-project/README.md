# Get Started

1. Change to the `test-project` directory:

    ```bash
    cd test-project
    ```

2. Install dependencies:

    ```bash
    npm install
 
    ```
3. Install node dddb driver
   
   Make sure to setup the driver locally before! ([`drivers/nodejs/dddb/README.md`](drivers/nodejs/dddb/README.md))
   
   ```bash
    npm link dddb
    ```

4. Start the server:

    ```bash
    node server.js
    ```
